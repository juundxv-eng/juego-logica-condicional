## 🎮 Descripción

**Enigma Code** es un juego educativo de descifrado de errores lógicos en pseudocódigo. 
Pon a prueba tus habilidades de programación descifrando y corrigiendo pequeños fragmentos 
de código antes de que el tiempo se agote.

## 🚀 Demo en Vivo

🔗 [Jugar ahora](https://juundxv-eng.github.io/juego-logica-condicional/)


## ✨ Características

- 🧠 5 niveles con errores lógicos progresivos
- ⏱️ Temporizador de 60 segundos
- 📊 Sistema de puntuación (100 puntos por nivel)
- 🎨 Interfaz estilo terminal con temática "encriptada"
- 📱 Diseño completamente responsive
- ✨ Animaciones y efectos visuales

## 🎮 ¿Cómo Jugar?

1. Lee el fragmento de pseudocódigo.
2. Identifica el error lógico.
3. Selecciona la corrección adecuada.
4. Gana puntos antes de que el temporizador llegue a cero.

---

## 🎓 Objetivo Académico

Este proyecto fue desarrollado como recurso didáctico para reforzar:

- Pensamiento computacional
- Lógica condicional
- Depuración de errores
- Análisis estructural de algoritmos
